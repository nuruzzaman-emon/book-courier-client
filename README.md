# Book Courier Web App <br/>

A modern book delivery and management web application built with React and Vite.

## Features

- User authentication
- User authorization
- Browse and order books
- Order tracking
- Dashboard for users and admins

## Tech Stack

- React
- Vite
- Tailwind CSS
- axios
- daisyui
- react leaflet
- lottie react
- motion
- react icon
- styled component
- sweetalert2
- swiper
- React Router
- TanStack Query
